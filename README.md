# BiliFans

> 超简单的 Bilibili 粉丝数显示 chrome 插件！

---

## Usage

输入 spaceid

### 如何获取 spaceid ?

> 打开 bilibili 个人空间

```js
"https://space.bilibili.com/14036737/#/";
//                         |________|
//                             ||
//                       这串数字就是辣。
```

### 点击扩展栏图标

还可以重新设置 spaceid.
